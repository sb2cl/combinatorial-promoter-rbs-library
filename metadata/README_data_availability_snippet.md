## Data and design availability

Annotated GenBank files for all library plasmids are provided in the GitHub repository under `genbank/`.
A machine-readable SBOL3 representation of the library design, including a formal `CombinatorialDerivation`
encoding promoter and RBS variation, is provided under `sbol/`.

A frozen release of the repository is archived on Zenodo (DOI to be added upon release). GenBank accession
numbers will be included in the manuscript once assigned.

### Library contents (summary)
- Backbones / replication origins: pSC101 (15 constructs), pGreen (21 constructs)
- Promoters: BBa_J23106, BBa_J23102, BBa_J23101, BBa_J23100
- RBSs: BBa_B0030, BBa_B0032, BBa_J61100, BBa_J61101, BBa_B0034
- Reporter: GFPmut3 (LuxR in control plasmid pARKA21)
